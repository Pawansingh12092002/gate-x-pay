# Gate X Pay

A modern fintech payment platform built using Next.js.

## 🚀 Features

- Authentication
- Responsive UI
- Fast loading
- SEO optimized

## 🛠️ Tech Stack

- Next.js
- Tailwind CSS
- React
- Vercel

## 📦 Installation

```bash
git clone REPO_URL
cd gate-x-pay
npm install
npm run dev
```
